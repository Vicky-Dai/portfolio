# Portfolio Website

A modern, professional portfolio website built with Next.js, React, TypeScript, and Tailwind CSS.

## Features

- 🎨 Modern, clean design
- 📱 Fully responsive
- ⚡ Fast and optimized
- 🌙 Dark mode support
- ✨ Smooth animations
- 🎯 SEO friendly

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

### Update Your Content

1. **Hero Section** (`components/Hero.tsx`):
   - Replace "Your Name" with your actual name
   - Update job title and tagline

2. **About Section** (`components/About.tsx`):
   - Add your professional summary
   - Update quick facts (location, education, etc.)

3. **Experience Section** (`components/Experience.tsx`):
   - Edit the `experiences` array with your work history
   - Add/remove experience items as needed

4. **Projects Section** (`components/Projects.tsx`):
   - Update the `projects` array with your projects
   - Add GitHub and live demo links

5. **Skills Section** (`components/Skills.tsx`):
   - Modify the `skillCategories` array with your skills
   - Organize skills by category

6. **Contact Section** (`components/Contact.tsx`):
   - Update email, LinkedIn, GitHub, and phone links
   - Replace placeholder contact information

7. **Navigation** (`components/Navigation.tsx`):
   - Update the site name in the navigation bar

8. **Metadata** (`app/layout.tsx`):
   - Update the page title and description

## Build for Production

```bash
npm run build
npm start
```

This portfolio can be easily deployed to:

- [Netlify](https://netlify.com)
- Any hosting service that supports Node.js
